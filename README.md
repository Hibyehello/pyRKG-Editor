# pyRKG-Editor
This is an gui editor to allow for easy editing of the metadata for Mario Kart Wii Time Trial Ghosts (RKG file format).

## Setup

```bash
git clone https://github.com/hibyehello/pyRKG-Editor.git
cd pyRKG-Editor
git submodule update --init

pip -r requirements.txt
```

# Running

```bash
python main.py
```
